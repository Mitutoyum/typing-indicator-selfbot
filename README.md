Discord SelfBot that spams the typing indicator on a specific channel with multiple account support

## Installation

```bash
git clone https://github.com/Mitutoyum/typing-indicator-selfbot.git
cd typing-indicator-selfbot/
```
Create a `config.json` under the directory, paste this template and change it to yours
```json
{
    "channel_id": CHANNEL_ID,
    "tokens": []
}

```
